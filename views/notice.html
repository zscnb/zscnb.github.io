电脑必会的知识点

vuePress-theme-reco Mr.Zhao    2019 - 2021

电脑必会的知识点

Mr.Zhao 2020-03-13 11:18:14 基础必会博客

掌握几项电脑常用技术，其中含有破解版技术(不是会员免费看，免费听)!! 🎉 对于拥有电脑一脉来说极为重要，现在有我来介绍下····

①. ➠ 电脑越用越卡： 用电脑搜索命令提示符，然后以管理员身份运行，在弹出框中输入CHKDSK,回车继续，再输入SFC/SCANNOW 回车结束

②.➠ 任务管理器磁盘占用满：点击电脑，然后点击管理，进入计算机管理，点击服务，找到Superfetch程序，点击它把它启动类型改为禁用

③ ➠ 开机只有一个鼠标：同时按住Ctrl + Alt + DEL,打开任务管理器，点文件，运行新任务，输入explorer,点击确定

④. ➠ 一招让你的网速飞起来：按下键盘win + R键 ,输入gpedit.msc,点击确定，点击windows设置，右键点击基于策略QoS,选择高级QoS设置，勾选上指定入站TCP吞吐量级别，选择级别3，点击确定，再点击管理模板，双击网络，找到QoS数据包双击打开，再双击打开可保留带宽，把未配置改为已启用，宽带限制改为零，点击确定。

⑤. ➠ C盘爆满怎么办：右键此电脑，选择属性，选择高级系统设置，选择高级，选择设置，点击高级，点击更改，把这个对勾去掉，勾选上无分页文件，点击设置，点击是，再选择D盘，勾选上系统管理的大小，点击设置，点击确定，点击确定。然后再重启电脑。

⑥. ➠ 开机时间慢怎么办：找到此电脑，右击找到属性，在这里有个高级系统设置，点击下方的设置，有一个显示操作系统列表时间和在需要恢复时间，把它都改为零就可以啦。

⑦. ➠ 一招让你U盘远离病毒：首先我们打开电脑，进入我们的U盘，在我们U盘里面新建一个文本文档autorun.inf,在点击更改好图标，进入属性，勾选只读和隐藏，点击应用和确定就可以啦。

⑧. ➠ 教你一个设置提升CPU性能的方法：首先win + R 键,输入msconfig, 点击确定。选择引导，点击高级选项，勾上处理器个数，把数值调到最大，点击确定应用，然后重启电脑即可啦！！

⑨. ➠ 4个电脑常用的技巧：

  • win + shift + s,可以快速使用系统自带的截屏工具
  • win+ + 键可以放大功能 ，win+ -键就是缩小功能
  • 想看电影电视剧但是没有会员，在网址最前面输入wn.run/,全网电影电视剧免费看
  • 网购买东西，但不知道是贵了还是便宜了，在网址最前面输入wn.run/,商品历史价格一目了然。

⑩. ➠ 快速永久关闭电脑弹窗广告：

  1. 多个禁用法：点击电脑 → 管理 → 任务计划程序 → 任务计划程序库 → 把右边程序选择一一禁用

  2. 单个禁用法： 点击广告原文件删除 → 又上一级(父级) → 点击父级文件夹 → 属性选择只读(禁止二次生成广告) ，注意权限即可 → 选择确认即可

  3. 点击控制面板 → Internet选项 → 点击隐私 → 启用弹出窗口阻止程序打勾 → 点击旁边的设置 → 把阻止级别改为高：阻止所有弹出窗口，你在上网时就不会弹出烦人的广告

  4. ★☆博主在用☆★首先win + R,打开运行窗口， 输入msconfig,进入系统配置窗口，点击引导，点击安全引导，调到最小，然后点击确定，然后重新启动电脑，就进入安全模式，把流氓软件卸载掉了，退出安全模式，用刚刚的方法，把安全模式去掉，点击应用，点击确定，重新启动了，就成功了卸载流氓软件，博主已经超过三周没有广告了！

  5. ➠ 电脑插入U盘提示格外化怎么办：

步骤：

  1. 首先win + R,打开运行窗口

  2. 输入cmd,打开管理员窗口

  3. 然后输入chkdsk H:/f,

  4. 注意H是我们U盘驱动器名，/f是修复命令

  5. ➠ 知道谁对你的电脑做了什么：

步骤：

  1. win + R 打开运行窗口
  2. 输入recent
  3. 右键修改日期，就知道谁对你的电脑做了什么 大家都在发大家都在发

13.➠ 如何一键创建多个文件夹：

步骤：

  1. 新建文本文档，输入md空格后面全部文件名，每个文件名用空格隔开然后保存

  2. 重命名将后缀名改为bat

  3. 然后双击打开bat文件，多个文件夹就创建好了！

  4. ➠ 破解邻居家WIFI的密码：

步骤：

  1. 按下win + R 打开运行窗口

  2. 输入netsh wlan show profiles,点击回车

  3. 重复这行代码netsh wlan show profiles "wifi名称" key=clear

  4. 点击回车，关键内容就是wifi密码了，不谢了！！！！

  5. ➠ 教你怎么删除C盘的垃圾：

步骤：

  1. 按下win + R 打开运行窗口

  2. 输入%TEMP%,点击回车

  3. 映入眼帘的是C盘的垃圾，按住Ctrl + A,右键点击删除即可

  4. 注意：一定没有任何程序打开使用以上步骤，否则后果自负！！！

  5. ➠ 教你怎么避免电脑自动安装软件：

步骤：

  1. 按下win + R 打开运行窗口
  2. 输入gpedit.msc,点击回车
  3. 依次点击管理模板，Windows组件，Windows Installer
  4. 点击禁止用户安装，选择点击已启用，然后将用户行为调整为隐藏用户安装
  5. 点击确定，电脑禁止自动软件安装就设置好了。赶快去试试吧！
